# git-browser

极简 Git 图形化桌面客户端，专注于代码浏览和日常 Git 操作。项目遵循简洁优先的设计理念：只满足日常开发 80% 的常见需求，不追求覆盖所有 Git 高级功能。

## 特性

- **三段式布局**：顶部标题栏 + 左侧文件树 + 右侧代码浏览 + 底部命令/按钮区
- **多标签页文件树**：文件列表 / 变更暂存区 / 收藏夹
- **两种工作模式**：
  - **命令模式**：直接输入 Git 命令，所有命令结果弹窗显示
  - **按钮模式**：常用操作预设按钮（提交历史、分支管理、拉取、推送等）
- **Git 功能**：
  - 文件状态颜色可视化（新增绿色、修改蓝色、删除红色、冲突橙色）
  - 代码浏览带语法高亮
  - 分支浏览与管理（切换、删除）
  - 提交历史查看
  - 支持切换 Git 仓库（点击标题栏路径）
- **便捷体验**：
  - 命令输入框自动聚焦，随时输入无需点击
  - 命令执行中自动锁定，防止重复提交
  - `git commit` 不带 `-m` 参数时弹窗提示输入提交信息
  - 同时捕获 stdout 和 stderr，`git push`/`git pull` 结果正常显示
  - 5 分钟超时保护，避免交互式命令卡住界面

## 技术栈

- **运行时**：Electron 桌面应用
- **框架**：React 18 + TypeScript + Vite
- **样式**：Tailwind CSS
- **Git 集成**：isomorphic-git + 原生 Git 命令执行
- **语法高亮**：Prism.js
- **打包**：electron-builder

## 开发

```bash
# 安装依赖
npm install

# 开发模式运行
npm run dev

# 生产构建
npm run build
```

## 项目结构

```
git-browser/
├── src/
│   ├── main/              # Electron 主进程
│   ├── renderer/          # React 渲染进程
│   │   ├── components/    # UI 组件
│   │   └── utils/         # 工具函数
│   └── shared/            # 共享类型定义
└── ui-design/             # UI 设计文档
```

## 设计理念

- **简洁优先**：只做 80% 用户常用的功能，保持界面干净
- **性能优先**：大文件懒加载，目录树虚拟化渲染
- **安全优先**：加密凭证存储，危险操作二次确认
- **跨平台**：支持 macOS 和 Windows
